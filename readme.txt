Sistema de gestión de invitados con control de acceso por QR, pensado para eventos empresariales, institucionales o sociales.

== Descripción ==

**Código8 – Event Check-in Manager** permite crear, importar y gestionar invitados a un evento directamente desde WordPress, con control de ingreso por código QR y vista optimizada para móviles.

Ideal para eventos gratuitos con registro previo o invitaciones personalizadas.

**Características principales:**
* Creación automática de invitados mediante importador CSV (nombre, organización, mesa, ticket, evento).
* Exportación de datos con estado de check-in y observaciones.
* Asignación de invitados a eventos mediante taxonomía (Ej: Evento 2025).
* Página de check-in protegida, configurable por evento (Ej: `/evento-2025/?ticket=1`).
* Interfaz móvil para escaneo con smartphones.
* Registro de usuario del staff que realizó el check-in y campo de observaciones.
* Filtros avanzados en la tabla de administración: por organización, mesa, estado o evento.
* Personalización visual para botones grandes y colores adaptados a móviles.

== Instalación ==

1. Subí el archivo `codigo8-event-checkin-manager.zip` desde el panel de administración de WordPress → Plugins → Añadir nuevo → Subir plugin.
2. Activá el plugin.
3. En el menú lateral aparecerá **“Invitados”**, donde podrás importar la lista, ver los invitados, filtrar y exportar.
4. Accedé al check-in mediante la URL de evento (por ejemplo `/evento-2025/?ticket=123`).
5. ¡Listo! Ya podés controlar los accesos de tu evento.
